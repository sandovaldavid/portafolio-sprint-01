Acerca del Proyecto

Este portafolio web ha sido desarrollado como parte de mi formación profesional en el programa Oracle Next Education (ONE) en colaboración con Alura Latam, aplicando las mejores prácticas de desarrollo web moderno.

Diseñado con un enfoque en la experiencia de usuario, este sitio muestra mi trayectoria profesional, habilidades técnicas y proyectos destacados. La estética minimalista y profesional refleja mi filosofía de trabajo: soluciones elegantes y funcionales.

Tecnologías Utilizadas

  • HTML5 Semántico
  • CSS3 Modular
  • JavaScript
  • Diseño Responsive
  • Accesibilidad Web
  • UI/UX Design

El código ha sido estructurado siguiendo principios de modularidad y mantenibilidad, con una arquitectura CSS basada en componentes para facilitar futuras extensiones y modificaciones del portafolio.

Este proyecto representa mi compromiso con la excelencia técnica y mi capacidad para crear soluciones web completas, desde el diseño hasta la implementación.
